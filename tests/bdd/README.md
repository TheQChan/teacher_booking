# BDD-тесты расписаний

1. Разрешения на создание слотов (учитель создаёт слот, студент — нет, пересечения запрещены).
2. Поведение расписания: студент может записаться на слот и освободить его, другой студент такого сделать не может.
3. Регистрацию: успешная регистрация, повторный e-mail и несовпадающие пароли.

## Запуск
```bash
cd teacher_booking/tests
pytest bdd/test_sessions_bdd.py
```
